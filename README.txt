MINI-SITE FOOT KORNER — SCRIPT LIVE TIKTOK SHOP

Fichiers :
- index.html : mini-site complet, prêt à publier.

Publication rapide sur Netlify :
1. Aller sur https://www.netlify.com/
2. Se connecter ou créer un compte.
3. Glisser-déposer le dossier footkorner-live-site ou le fichier index.html.
4. Netlify génère un lien public partageable.

Publication sur GitHub Pages :
1. Créer un dépôt GitHub.
2. Envoyer index.html à la racine du dépôt.
3. Aller dans Settings > Pages.
4. Choisir Deploy from branch > main > root.
5. Copier le lien généré.
