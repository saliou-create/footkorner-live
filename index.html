Mode d'emploi

Utilise ce mini-site comme conducteur live : clique sur chaque créneau, lis le script, présente les produits, puis passe au bloc suivant.

1. PréparerValider les prix flash, stocks, tailles et produits giveaway.
2. OuvrirLancer cette page sur téléphone, tablette ou PC avant le live.
3. AnimerSuivre l'heure, lire les phrases clés, relancer les commentaires.
4. DébrieferNoter ventes, questions, problèmes techniques et produits performants.

Déroulé interactif

Chaque bloc est cliquable. Le premier bloc est ouvert par défaut pour démarrer rapidement.

⚠️ Les giveaways / enchères sont à confirmer avec Foot Korner avant le live. Les produits Under Armour doivent être préparés côté marque.
